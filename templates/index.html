Parking Lot Pathfinder — Web UI

Legend & Rules

  • . : Road (only walkable tile)
  • > < ^ v : Directional road tiles (one-way)
  • C : Car starting position
  • O : Lobby (not a through-road)
  • N : Naik (ramp up) — must have E above
  • T : Turun (ramp down) — must have e below
  • P/L/D : Occupied slot (blocked)
  • p/l/d : Empty slot (can be entered only as goal)
  • Paths are shown on the map as arrows (← → ↑ ↓). Lobby/entrance/slots are not used as roads.
Maps are loaded from the `maps/` folder in the project.
